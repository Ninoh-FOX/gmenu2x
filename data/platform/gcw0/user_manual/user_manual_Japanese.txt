特別なキーの組み合わせ：
----
POWER = Gmenu2xで画面をオンまたはオフにします。
POWER + A =ハードウェアモードのときに、フルスクリーンモードで画面解像度を変更します。
POWER + B =左スティックはDpadにマッピングされています。
POWER + X =スクリーンショットを撮ります。 （スクリーンショットに保存されています）。
POWER + L1 =アナログ/秒を非アクティブ化または再アクティブ化します。
POWER + R1 =マウスモードをアクティブにします（スティックはカーソル（RG350 / Mでは右スティック）、L2およびR2はマウスボタンです）。
注：RG280V POWER + L1 =マウスモードをアクティブにします（Dpadはカーソル、A＆Bはマウスボタンです）。
POWER + VOL +またはVOL- =画面の明るさを変更します。
POWER + UPまたはDOWN DPAD =画面の鮮明度を変更する
POWER + SELECT =アクティブなアプリケーションを閉じます。
POWER + START =コンソールを再起動します。
