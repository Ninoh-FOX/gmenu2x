Специальные комбинации клавиш:
----
POWER = включить или выключить экран в Gmenu2x.
POWER + A = Изменяет разрешение экрана в полноэкранном режиме, когда оно находится в аппаратном режиме.
POWER + B = левый джойстик отображается на Dpad.
POWER + X = Сделать скриншот. (сохранено на скриншотах).
POWER + L1 = Деактивирует или повторно активирует аналог / ы.
POWER + R1 = активировать режим мыши (ручка - курсор (правая ручка на RG350 / M), L2 и R2 - кнопки мыши).
Примечание: RG280V POWER + L1 = Активировать режим мыши (Dpad - это курсор, A&B - это кнопки мыши).
POWER + VOL + или VOL- = Изменяет яркость экрана.
POWER + UP или DOWN DPAD = Изменить резкость экрана
POWER + SELECT = Закрыть активное приложение.
POWER + START = Перезагрузить консоль.
